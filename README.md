Punjab Census Data Analysis

** Project Overview **

From time to time, official counts reveal how people are spread out by age, sex, and jobs in different areas. Within this work, information from each district of Punjab went through processing in Python, uncovering trends tied to society and economy.
Around 20 to 30 variables appeared at the start, though only a subset was ultimately used after selection and refinement. Patterns across districts emerged through methods like EDA, statistics, and machine learning - each playing a distinct role. While some features dropped away early, others transformed to better suit the modeling process. Insights developed gradually, shaped by both data structure and analytical choices made along the way.


** Objective **

To analyze census data and identify patterns in population, sex ratio, and workforce participation across districts.

** Tools & Technologies **
      Pandas (data manipulation) 
      NumPy (numerical operations) 
      Matplotlib & Seaborn (visualization) 
      Plotly (interactive dashboard) 
      Scikit-learn (machine learning) 

** Key Analysis **
     Population growth trends (1991–2011)
     Gender analysis (Sex Ratio)
     Workforce participation analysis
     Rural vs Urban population comparison
     etc..

** Machine Learning **
      Linear Regression (baseline model)
      K-Means Clustering (district grouping)

** Key Insights **
      Population growth is uneven across districts
      Average sex ratio (~896) indicates gender imbalance
      Workforce participation varies significantly
      No single factor determines development

** Dashboard **
An interactive dashboard was created using Plotly to visualize population distribution, growth trends, and workforce patterns.
